# ✊✋✌️ Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

This project allows users to play against the computer while practicing JavaScript logic, DOM manipulation, and front-end web development concepts.

---

## 🚀 Features

- Play Rock, Paper, Scissors against the computer
- Random computer choice generation
- Score tracking
- Win, Lose, and Draw detection
- Sound effects for game actions
- Responsive and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎮 How to Play

1. Choose Rock, Paper, or Scissors.
2. The computer will randomly make its choice.
3. The winner is decided using standard game rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
4. The score updates automatically.
